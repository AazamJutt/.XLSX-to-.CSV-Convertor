.XLSX to .TXT Convertor

Description

This tool takes and (.XLSX) input file and convert its data to (.TXT)
in Comma-Delimited Format. Default extension is (.TXT) but you can select (.CSV)

Instructions

• Add an (.XLSX) file by clicking, "Choose a file" button.
• Click the "Download Converted" button to download your converted file.
• If you Check the checkbox, File will be downloaded with (.CSV) extension.
